job-tracker-cli

Log job applications to Google Sheets from the command line.

job-tracker-cli is an open-source command-line tool that lets you track job applications in a Google Sheet with a single command. Paste a job posting URL and it auto-detects the company and title, then appends a row to your spreadsheet. 

job https://boards.greenhouse.io/acme/jobs/12345

Features

  • Auto-detects company and job title from Greenhouse, Lever, Ashby, Workday, and more
  • Appends rows to any Google Sheet you configure
  • Configurable columns, date format, and target tab
  • One-time Google OAuth setup — token stored locally

Install

pipx install git+https://github.com/CalvinFronda/job-tracker-cli
